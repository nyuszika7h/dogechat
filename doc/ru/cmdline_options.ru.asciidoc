*-a*, *--no-connect*::
    Отключить автоподключение к серверу при старте DogeChat.

*-c*, *--colors*::
    Вывести в терминал умолчальные цвета.

*-d*, *--dir* '<путь>'::
    Установить <путь> как домашнюю директорию для DogeChat (используется для
    конфигурационных файлов, логов, пользовательских плагинов и скриптов).
    Значение по-умолчанию: '~/.dogechat' (замечание: директория будет создана,
    если DogeChat не найдет ее).
    Если эта опция не задана, будет использована переменная окружения DOGECHAT_HOME
    (если такая переменная задана).

*-h*, *--help*::
    Вывести справку.

*-l*, *--license*::
    Вывести лицензию DogeChat.

*-p*, *--no-plugin*::
    Отключить автозагрузку плагинов.

*-r*, *--run-command* '<команда>'::
    Запустить команду (или команды) после загрузки DogeChat (несколько команд
    можно записать через точку с запятой).

*-s*, *--no-script*::
    Отключить автозагрузку скриптов.

*--upgrade*::
    Обновить DogeChat с использованием сеансовых файлов, сгенерированных
    командой `/upgrade -quit`.

*-v*, *--version*::
    Вывести версию DogeChat.

*<плагин>:<параметр>*::
    Задать параметр плагина.
